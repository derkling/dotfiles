## Gemini Added Memories
- Markdow files textual content is always reflowed to fit into 80 columns.
- Trailing whitespaces are always removed from code.
- Git commit messages are always 78 cols reflowed and provide a changelog
  describing the issue, an empty line, and the proposed solution. Use an
  imperative tone. Do not start with explicit Problem/Solution words, be
  descriptive.
